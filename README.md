# TD7 Photobox

Projet réalisé en TypeScript dans le cadre du TD7 de JavaScript.

## Fonctionnalités réalisées

### Exercice 1
- Chargement d’une photo depuis l’API
- Affichage :
  - de l’image
  - du titre
  - de la description
  - de la catégorie
  - des commentaires

### Exercice 2
- Affichage d’une galerie de photos

### Exercice 3
- Navigation dans les pages de la galerie :
  - first
  - prev
  - next
  - last

### Exercice 4
- Affichage de la photo complète lors du clic sur une vignette

### Bonus
- Ajout d’une lightbox pour afficher les photos
